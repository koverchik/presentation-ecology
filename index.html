Оформление и структура приложения

При загрузке приложения вы попадаете на главную страницу. Визуально экран приложения разделен на 2-е половины. В левой части экрана расположена интерактивная навигационная карта, справа информационный блок с интерактивными элементами.
В верхнем левом углу находится кнопка главного меню. В правом углу кнопка входа.

Функционал

Меню станет доступно после нажатия кнопки главного меню.
По кнопке присоединиться пользователь попадет на страницу «Карта мероприятий»
По кнопке «Вход» пользователя отправит на страницу регистрации, если пользователь уже вошел или зарегистрировался, кнопка входа будет заменена на кнопку профиля и при клике будет направлен на страницу профиля.

Страница карты мероприятий

Карта мероприятий показывает место в котором проведения мероприятий, которые визуально группируются на прошедшие, текущие и запланированные.

Легенда карты - описание цветового соответствия находится в левом верхнем углу карты.

Функционал карты мероприятий

Карта отображает мероприятия за все время. При клике по мероприятию появится подсказка с информацией о мероприятии.

Функционал карты мероприятий

Календарь мероприятий

Календарь мероприятий содержит название мероприятий на определенную дату.

Календарь отображает мероприятия за месяц, при клике по названию мероприятия можно посмотреть информацию о данном мероприятии.

Список мероприятий

Список мероприятий на выбранную дату отображается под Календарем мероприятий.

В данном блоке отображается список мероприятий на выбранную дату в календаре.

Страница мероприятия

При клике на ссылку «Посмотреть» открывается страница с описанием мероприятия с возможностью присоединиться к данному мероприятию нажав кнопку «Присоединиться».

Если мероприятие было создано самим пользователем, то он будет направлен на страницу «Создания нового мероприятия».

Страница мероприятия

Страница создания мероприятия

Отображает карту мероприятий и блок для создания нового мероприятия пользователем.

В данном блоке пользователь заполняет информацию о новом мероприятии. После нажатия по кнопке создать мероприятие появится на карте и в календаре. Поля страницы имеют валидацию для обеспечения корректности введенных данных.

Если пользователь не был зарегистрирован, при нажатии на данный пункт меню он будет перенаправлен на страницу регистрации.

Страница создания мероприятия

Игра

Страница с игрой просит нажать кнопку Play, после чего отобразится приветственный экран. Далее следует страница со списком доступных уровней. После выбора уровня начинается игра.

Процесс игры (функционал)

Суть игры заключается в распределении мусора по соответствующим контейнерам путем перетаскивания при удержании кнопки мыши, если предмет не соответствует контейнеру предмет не будет помещен в контейнер.

При успешном окончании игры появится поздравительное сообщение, есть возможность выйти из игры не закончив уровень.

В игровом меню доступна кнопка отображающая правила игры.

На странице доступна информация о пользователе, его достижения в виде единиц кислорода, истории событий которая отображает действия пользователя.

Организация командной работы

При разработке приложения RACI матрица не использовалась, так как в разработке проекта участвовало 3-и человека и данный инструментарий был избыточным.

Инструменты для организации совместной работы

  • Trello для распределения задач
  • Discord для ведения совместных встреч
  • Telegram для информирования и прояснения вопросов возникших в ходе работы
  • Github для совместной разработки
  • Figma для создания макета приложения
При возникновении сложностей у одного из участников команды, организовывался внеплановый митинг, на котором методом мозгового штурма предлагались возможности решения проблемы.
